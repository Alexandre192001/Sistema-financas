<h2>⚜️Descriçao⚜️</h2>
<p>Projeto desenvolvido para gerenciamento de finanças, colocando a descrição, valor e o tipo se é crédito ou débito e vai armazenando esses dados
localmente, ou seja, sempre que precisar só é consultar o programa que vai está salvo no navegador local.</p>
<hr>
<h2>⚜️Funcionalidades:⚜️</h2>
<ul>
<li>Descrição - Informar o nome da produto;</li>
<li>Valor -  Informar qual o valor do produto;</li>
<li>Entrada/saída - colocar qual o tipo de ação, se é de entrada ou saída</li>
<li>Adicionar - Vai adicionar na tabela esse produto e logo acima será possível ver o total de entrada/saída e o valor total.</li>
</ul>
